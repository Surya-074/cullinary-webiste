# Fridge Hero 🍳

A modern recipe app that magically creates recipes from your fridge ingredients.

## Features

- Enter up to 3 ingredients
- Click "Cook Magic" for AI-generated recipes
- Modern UI with shadows and rounded corners

## Getting Started

1. Install Node.js (LTS version)

2. Install dependencies:
   ```
   npm install
   ```

3. Run the development server:
   ```
   npm run dev
   ```

4. Open http://localhost:5173 in your browser

## Build

```
npm run build
```
