# Gustavo's Portfolio

A modern, colorful, and creative personal portfolio website built with HTML, CSS, and JavaScript.

## Features

✨ **Modern Design**
- Vibrant gradient color scheme
- Smooth animations and transitions
- Responsive design (mobile, tablet, desktop)
- Interactive elements

🎯 **Sections Included**
- **Hero Section** - Eye-catching introduction
- **About** - Personal bio with stats
- **Projects** - 6 featured project cards with tags
- **Skills** - Categorized technical skills
- **Contact** - Contact form and information

⚡ **Functionality**
- Smooth scroll navigation
- Mobile-responsive hamburger menu
- Contact form
- Scroll-triggered animations
- Active navigation highlighting

## Getting Started

1. **Open the Portfolio**
   - Simply open `index.html` in your web browser
   - No installation or dependencies required

2. **Customize Your Content**
   - Edit `index.html` to update personal information
   - Replace placeholder text with your actual details
   - Add your own contact information
   - Update project descriptions and links

3. **Personalize Styling**
   - Modify colors in `styles.css`
   - Adjust fonts and spacing as needed
   - Change gradient colors to match your brand

## Customization Guide

### Update Personal Information
In `index.html`:
- Change "Gustavo" to your name in multiple places
- Update email and phone in contact section
- Modify social media links
- Update project descriptions

### Modify Project Cards
Replace the 6 sample projects with your own:
- Update project titles
- Change project descriptions
- Modify the gradient colors for project images
- Update technology tags

### Change Color Scheme
The main gradient colors used throughout:
- Primary: `#667eea` to `#764ba2` (purple/blue)
- Accent: `#ffd89b` to `#19547b` (yellow/dark blue)

To change globally, search-replace these values in `styles.css`

### Add Your Social Links
In the contact section, update:
```html
<a href="https://github.com/your-username" class="social-link">GitHub</a>
<a href="https://linkedin.com/in/your-profile" class="social-link">LinkedIn</a>
<a href="https://twitter.com/your-handle" class="social-link">Twitter</a>
```
